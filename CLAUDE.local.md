# 코드 스타일

- [중요] 들여쓰기 : 스페이스 2칸
- 세미콜론 사용
- 작은따옴표('') 사용하지 않음

# 언어 및 커뮤니케이션 규칙

- 기본응답언어 : 한국어
- 코드주석 : 한국어로 작성
- 문서와 한국어로 작성
- 변수명/함수명 : 영어, camel표기법 (코드표준준수)
- css명 : 영어, camel표기법 (코드표준준수)
- 주석은 css는 제외 하고 왜 그렇게 사용했는지 항상 한글로 자세히 작성해줘
- rem 변경 주석은 작성하지 마세요
- css는 어디 컨텐츠인지만 기록해주세요
- css 모든 컬러는 변수로 작성하지말고 hex 코드로 작성해줘
ex) #555;"
- css 간격은 clamp(20px, 6vw, 40px)로 작성하지말고 rem으로 작성해줘
ex) gap: 120rem; , padding:2rem 4rem " 
- 로고를 h1부터 시작해서 시멘틱 마크업으로 작업해줘
